# zzd
Basic clone of xxd to take a look at zig.

```
$ zzd --help
    -h, --help
            Display this help text.
    -i, --input <str>
            Input file path.
    -c, --cols <u32>
            Number of bytes per line.
    -g, --bytes <u32>
            Number of bytes per group.
    -s, --skip <usize>
            Start printing at that offset.
    -l, --limit <usize>
            Stop printing after that amount of bytes.
```
